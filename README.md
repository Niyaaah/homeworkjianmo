# UPPAAL 电梯系统建模与验证

本项目使用 UPPAAL 对电梯系统的三个子系统进行建模和验证。

## 项目结构

- `passenger.xml` - 乘客子系统模型
- `elevator.xml` - 电梯子系统模型
- `scheduler.xml` - 调度器子系统模型
- `integrated_system.xml` - 完整集成系统模型

## 子系统说明

### 1. 乘客子系统
- 状态：空闲、等待电梯、乘梯中、离开电梯、放弃等待
- 核心行为：等待 - 乘梯 - 离开

### 2. 电梯子系统
- 状态：待机、上行运行、下行运行、停靠中、故障暂停
- 核心行为：运行 - 停靠 - 待机

### 3. 调度器子系统
- 状态：监听中、调度决策、指令跟踪、应急调度
- 核心行为：请求处理 - 指令分发 - 状态监控

## 使用说明

1. 安装 UPPAAL 软件
2. 打开对应的 .xml 文件
3. 运行仿真和验证

## 验证性质

- 安全性：系统不会进入错误状态
- 活性：所有请求最终会被处理
- 无死锁：系统不会死锁
